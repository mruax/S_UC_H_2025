# Main

## Карта портов системы "Динамический планировщик образовательной траектории"

Система использует следующие порты для избежания конфликтов между Moodle Docker и основным приложением:

### Moodle Docker (приоритетные порты - не изменять)
| Сервис | Внешний порт | Внутренний порт | Описание |
|--------|--------------|-----------------|----------|
| Moodle Web | 8000 | 80 | Веб-интерфейс Moodle |
| PostgreSQL (Moodle) | 5432 | 5432 | База данных Moodle |
| Redis (Moodle) | 6379 | 6379 | Кеш Moodle |

### Основное приложение (изменённые порты для избежания конфликтов)
| Сервис | Внешний порт | Внутренний порт | Описание |
|--------|--------------|-----------------|----------|
| Backend API | 8080 | 8080 | REST API сервис |
| PostgreSQL (App) | 5433 | 5432 | База данных приложения (изменён с 5432) |
